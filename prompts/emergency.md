# 应急响应 Prompt / Emergency Response Prompt

## 任务 / Task

在紧急情况下，快速生成简洁有效的甩锅话术。
In emergency situations, quickly generate concise and effective buck-passing话术.

## 紧急情况分类 / Emergency Categories

### 🔥 类型1: 当场质问 / On-the-spot Questioning
**场景**: 会议上突然被质问
**要求**: 3秒内组织语言

### ⚡ 类型2: 群聊@ / Group Chat Mention
**场景**: 在群里被@要求解释
**要求**: 文字回复，有时间思考但不多

### 🚨 类型3: 上级突击 / Superior Ambush
**场景**: 老板突然找你谈话
**要求**: 面对面快速反应

### 💥 类型4: 邮件轰炸 / Email Bombardment
**场景**: 收到质问邮件，需要回复
**要求**: 书面回复，可以稍作准备

## 快速话术模板 / Quick话术 Templates

### 当场质问 - 缓冲型
```
"这个问题我需要核实一下具体细节，稍后给您准确回复。"
"I need to verify the specific details on this and will get back to you shortly."
```

### 当场质问 - 转移型
```
"这个情况我了解一部分，[同事名]可能掌握更全面的信息。"
"I have partial information on this; [colleague name] may have the full picture."
```

### 群聊@ - 拖延型
```
"收到，我正在查相关记录，10分钟后同步。"
"Got it, checking relevant records now, will update in 10 minutes."
```

### 群聊@ - 澄清型
```
"这个部分我当时是按要求执行的，具体决策背景可能需要确认一下。"
"I executed this part as requested; the decision context may need verification."
```

### 上级突击 - 缓冲型
```
"您提到的这个问题，我需要整理一下时间线，这样汇报更准确。"
"Regarding this issue, I need to organize the timeline for a more accurate report."
```

### 上级突击 - 担当型（反向）
```
"这个问题我有责任，当时应该更主动地同步风险。"
"I bear responsibility for this; I should have proactively communicated risks."
```

### 邮件轰炸 - 标准回复
```
Subject: Re: [原主题]

感谢您的反馈。

经过核实，[简要说明情况]。

当时的决策背景是[背景说明]。

建议我们[建设性提议]。

Best regards,
[你的名字]
```

## 万能金句 / Universal Golden Lines

### 争取时间
- "我需要确认一下细节 / I need to confirm the details"
- "让我查一下记录 / Let me check the records"
- "这个问题比较复杂，我需要整理一下思路 / This is complex, let me organize my thoughts"

### 转移焦点
- "这个问题涉及多个环节 / This involves multiple stages"
- "当时的决策是基于... / The decision was based on..."
- "我们需要从更宏观的角度来看 / We need to look at this from a broader perspective"

### 缓和气氛
- "我理解大家的担忧 / I understand everyone's concerns"
- "这个问题确实需要重视 / This issue does need attention"
- "我们一起看看怎么解决 / Let's look at how to solve this together"

## 应急流程 / Emergency Process

### Step 1: 判断情况 (1秒)
- 对方情绪状态 / Opponent's emotional state
- 场合正式程度 / Formality of the occasion
- 证据对你是否有利 / Whether evidence favors you

### Step 2: 选择策略 (1秒)
- 有利 → 证据型 / Favorable → Evidence-based
- 不利 → 缓冲型 / Unfavorable → Buffering
- 不确定 → 拖延型 / Uncertain → Delaying

### Step 3: 组织语言 (1秒)
- 使用模板 / Use template
- 调整关键词 / Adjust keywords
- 保持冷静 / Stay calm

## 禁忌 / Taboos in Emergency

❌ 不要直接说"不是我的错" / Don't directly say "It's not my fault"
❌ 不要指责具体人名 / Don't name specific people
❌ 不要情绪化 / Don't get emotional
❌ 不要编造证据 / Don't fabricate evidence
