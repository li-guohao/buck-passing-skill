# 甩锅 Skill / Buck-Passing Skill

## 核心原则 / Core Principles

1. **优雅得体 / Graceful**: 不直接否认，而是转移焦点
2. **证据导向 / Evidence-based**: 用数据和事实说话
3. **团队协作 / Team-oriented**: 强调共同责任而非个人指责
4. **建设性 / Constructive**: 指向解决方案而非问题本身

## 使用场景 / Use Cases

- 项目延期需要解释 / Project delay explanations
- 需求变更引发的问题 / Issues from requirement changes
- 跨部门协作不畅 / Cross-department collaboration issues
- 资源不足导致的失败 / Failures due to insufficient resources
- 上级决策失误 / Superior decision-making errors

## 甩锅等级 / Buck-Passing Levels

### Level 1 - 轻微转移 / Slight Deflection
适用于小问题，保持和谐关系

### Level 2 - 合理澄清 / Reasonable Clarification
明确责任边界，但不伤和气

### Level 3 - 坚决自保 / Firm Self-Protection
涉及重大利益时的强硬策略

### Level 4 - 绝地反击 / Counter-Attack
当对方先甩锅时的反击策略
