# 重要声明 / Important Disclaimer

## 项目定位 / Project Positioning

**甩锅-skill** 是一个**职场沟通技巧学习工具**，旨在帮助用户：
- 理解职场责任归属的复杂性
- 学习如何在保护自己权益的同时维护团队和谐
- 掌握非暴力沟通技巧

**Buck-Passing Skill** is a **workplace communication skills learning tool** designed to help users:
- Understand the complexity of workplace responsibility attribution
- Learn how to protect their rights while maintaining team harmony
- Master non-violent communication techniques

---

## ⚠️ 正确使用方式 / Correct Usage

### ✅ 鼓励使用场景 / Encouraged Scenarios

1. **澄清误解** / Clarifying Misunderstandings
   - 当责任被错误归因时，有理有据地说明情况
   - When responsibility is wrongly attributed, explain the situation with reason and evidence

2. **保护团队** / Protecting the Team
   - 防止团队成为不合理指责的受害者
   - Prevent the team from becoming victims of unreasonable accusations

3. **促进沟通** / Facilitating Communication
   - 用建设性方式指出系统性问题
   - Point out systemic issues in a constructive manner

4. **学习分析** / Learning Analysis
   - 理解职场冲突中的责任分配逻辑
   - Understand the logic of responsibility allocation in workplace conflicts

### ❌ 禁止滥用 / Prohibited Misuse

1. **恶意推卸责任** / Malicious Responsibility Evasion
   - 明明是自己的错误却强行甩锅给他人
   - Forcing blame onto others when it's clearly your own mistake

2. **破坏团队信任** / Destroying Team Trust
   - 用话术技巧逃避应尽的义务
   - Using话术 techniques to evade obligations

3. **陷害同事** / Framing Colleagues
   - 故意将责任转嫁给无辜的同事
   - Deliberately shifting blame to innocent colleagues

4. **逃避成长** / Avoiding Growth
   - 用"甩锅"代替反思和改进
   - Using "buck-passing" instead of reflection and improvement
